# 스크린 리더 호환성

일부 플레이어는 스크린 리더를 사용하여 Cataclysm Bright Nights를 플레이합니다. 스크린 리더가 UI의 주요 정보를 올바르게 읽어주려면 터미널 커서가 적절한 위치에 있어야 합니다. 예를 들어 목록에서 선택된 아이템 이름 등의 텍스트를 읽으려면, 커서가 해당 텍스트의 시작 지점에 정확히 위치해야 합니다.

커서 위치를 설정하는 권장 방법은 `ui_adaptor`를 사용하는 것입니다. 이렇게 하면 이후의 출력 코드가 커서 위치를 변경하더라도 원하는 커서 위치가 유지됩니다. 재그리기 콜백 내 어디서든 `ui_adaptor::set_cursor` 및 관련 메서드를 호출할 수 있으며, 최상위 UI의 마지막 커서 위치가 최종 커서 위치로 사용됩니다. 또한 `ui_adaptor::disable_cursor`를 호출하여 특정 UI의 커서가 최종 커서 위치로 사용되지 않도록 할 수 있습니다.
