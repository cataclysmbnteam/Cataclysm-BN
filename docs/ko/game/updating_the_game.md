# 게임 업데이트하기

# 올바른 방법

0. https://github.com/cataclysmbn/Cataclysm-BN/releases 에서 게임 받기
1. 게임 압축 풀기
2. 이전 게임 디렉토리에서 새 디렉토리로 `save`와 `config` 복사
3. (선택사항) 이전 게임 디렉토리에서 `mods` 복사

# 하면 안 되는 것

### 이전 디렉토리에 새 게임 압축 풀기

이전 디렉토리에 게임을 압축 풀면 중복 JSON 항목 오류가 발생할 수 있습니다. 정말 그렇게 하고 싶다면, 새 버전을 압축 풀기 전에 이전 `data` 디렉토리를 삭제하세요.

때때로 코어에서 데이터 파일이 삭제됩니다. 새 버전의 게임에는 그런 파일이 없지만, 아카이브 압축 풀기는 이전 파일을 삭제하지 않습니다. 이전 파일이 여전히 읽히고 로드되어 새 항목을 덮어쓸 수 있습니다.

### `data/mods` 디렉토리에 커스텀 모드 두기

`data`와 같은 레벨의 `mods` 디렉토리에 직접 넣으세요. 이 디렉토리는 기본적으로 존재하지 않지만, 있으면 게임이 읽습니다.

이렇게 하면 게임 버전 간에 모드를 복사할 수 있으면서도 "코어 모드"가 제대로 업데이트될 수 있습니다.
