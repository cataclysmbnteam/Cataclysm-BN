# ゲームの更新

# 推奨事項

0. https://github.com/cataclysmbn/Cataclysm-BN/releases からゲームを入手する
1. ゲームを解凍する
2. 古いゲームディレクトリから新しいディレクトリへ `save` と `config` をコピーする
3. (任意) 古いゲームディレクトリから `mods` をコピーする

# 非推奨事項

### 古いディレクトリに新しいゲームを上書き解凍する

古いディレクトリに上書き解凍すると、JSONエントリの重複エラーが発生する可能性があります。どうしても行いたい場合は、新しいバージョンを解凍する前に古い `data` ディレクトリを削除してください。

コアデータファイルが削除されることがあります。新しいバージョンのゲームにはそれらのファイルが含まれていませんが、アーカイブを解凍しても古いファイルは削除されません。古いファイルが読み込まれ、新しいエントリが上書きされる可能性があります。

### `data/mods` ディレクトリにカスタムMODを配置する

`data` と同じ階層にある `mods` ディレクトリに直接配置してください。このディレクトリはデフォルトでは存在しませんが、存在すればゲームは読み込みます。

これにより、ゲームバージョン間でMODをコピーしつつ、「コアMOD」を適切に更新することができます。
