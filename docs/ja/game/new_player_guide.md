# 新規プレイヤーガイド

> [!TIP]
>
> キー操作で混乱した場合は、いつでも `?` を押してゲーム内のキー割り当てを確認するか、
> メインメニューからキー割り当てメニューを表示してください。

## 最初のキャラクター

冒険は突然終わることがあります。避難シェルターでデフォルトの生存者として始めると、かなりストレスのない環境でゲームを学ぶことができます。ソーラーパネルとバッテリーがあるので、オーブンや冷蔵庫を設置するのにほとんど手間がかかりません。森を利用して食料と水を確保し、町に行く前にスキルを鍛えることができます。木槍や何らかの遠距離攻撃は、初期のゾンビを倒すためのかなり安全な方法です。森だけで完全に自給自足することもできますが、モンスターの進化に追いつくための競争になります。平均的な銃砲店にある銃は、生き残ってそこに入ることができれば十分です。アクセスできない場所に遭遇した場合、つるはしや削岩機でコンクリートや金属の壁やドアを破壊でき、弓のこで窓の鉄格子を切断できます。戦闘ではスタミナ管理が重要です。スタミナが尽きると、特に新規プレイヤーの場合はおそらく死ぬでしょう。

### 製作

中世の装備を鍛造するために製作スキルを上げていくことができます。近接キャラクターとして、これは重要になります。町で図書館を見つけると、このスキルレベルまで読むために必要な本を見つけることができますが、レシピのためにどうしても本が必要になります。銃器や近接戦闘でゾンビがほとんど脅威でなくなったら、他のスキルを訓練し、動く車両を見つけ、より危険で報酬の高い場所を略奪する準備をすることを検討したくなるでしょう。

### CBMや変異原物質の発見

CBMにはオートドックが必要です（手動インストールModが有効でない場合）。インストールを成功させるには、高い応急手当、コンピュータ、電子工作スキルが必要です。手動インストールにはコンピュータの代わりに機械整備が必要です。一部のゾンビはCBMを持っており、メスと高い応急手当スキルで解剖するとこれらが得られます。変異システムについてあまり多くを語らずに言うと、このルートを追求するには研究所を略奪したくなるでしょう。タレットは非常に危険で、冒険をすぐに終わらせる可能性がありますが、見つかるのはそれだけではありません。科学IDを持っていない限り、ハッキングするか、力ずくで侵入することになります。

## 自動アクション

自動アクションを使用すると、キーストロークを最小限に抑えることができます。

### 自動拾得

薬莢やショットガンシェルを自動で拾うように設定できます。重量や体積の制限、戦闘中に拾うのを避けるためのセーフモードなど、自動拾得を適切に利用することが重要です。

### 自動解体（死体破壊）

自動解体を使用すると、キルごとに `(s)` を押して死体を破壊することなく、戦闘中に死体解体を行うことができます。

### 自動採集

自動採集は自給自足生活に最適ですが、移動中は無効にすることを忘れないでください。

### 自動採掘

自動採掘を使用すると、地形に繰り返し移動して採掘できます。自動採掘は危険です。採掘したくない壁に誤って歩いてしまったり、つるはしを武器として使用している場合、意図せず拠点を破壊してしまう可能性が非常に高いです。

## ゾーンマネージャー

ゾーンマネージャー `(Y)` を使用すると、戦利品を自動分類したり、地形や車両を自動的に解体したり、建設設計図を作成したり、魚を養殖したり、木を伐採したりするための分類ゾーンを設定できます。これはキーストロークを最小限に抑えるための強力なツールです。

### ゾーンアクティビティ

`(O)` を使用すると、少なくとも1つのゾーンがあれば、キャラクターはゾーンアクティビティを実行できます。

### 分類

地面にあるものを分類し始めるには、少なくとも1つの未分類戦利品ゾーンが必要です。家全体を未分類戦利品ゾーンに設定することができ、それらは重複しても構いません。車両の収納タイルにゾーンを使用できますが、複数のタイルにまたがることはできず、車両内への分類も可能です。未分類戦利品ゾーン内でのみ分類を行い、インベントリ内では行いません。ただし、戦利品を分類するために空きインベントリスペースを使用するため、バッグは空けておいてください。

## ヒントとよくある質問

1. インベントリ内で `TAB` を使用してカテゴリ全体を選択できます。例：`d` でマルチドロップを使用してすべての食料を一度に落とす。
2. キーの再バインドは、特にWASDでプレイする場合、キー入力を大幅に改善できます。
3. セーフモードを使用してください。モンスターを無視できますし、あまりにも気になる場合は近接距離を適切な値に設定してください。セーフモードはあなたを救います。
4. 製作範囲、スキルトレーニング速度、または実験的な3D視界を変更したい場合は、オプションのデバッグカテゴリで見つけることができます。
