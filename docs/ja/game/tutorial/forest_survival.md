# 森林サバイバル

Cataclysmの最も難しい部分でありながら、何もない森で生き残ることは最もやりがいのある経験の1つです。このガイドでは、あなたを生かし続けるためのタスクリストをカバーします。遅れをとった場合は、次の日のタスクに進む前に前のステップを続けてください。例外は1日目です。1日目のタスクは深刻度の順になっています。

> [!CAUTION]
>
> このガイドは、これをあまりにも変更するMOD、特に採集ドロップを変更するMODで遊んでいないことを前提としています。低木からゴミが削除されている場合、あなたは死ぬでしょう。

## 1日目の要件

- 火起こし器（サバイバル1）または任意の着火具
- 火囲い（石20個）
- 水源と調理容器
- 簡易シェルター（サバイバル2）、さもないと睡眠は困難になるでしょう

### 火

火が必要な理由は、夜にクラフトするための視界がなく、寒さが命取りになるからです。火なしでは水を沸騰させることはできません。火囲いなしで火を使いたくはないでしょう。それは信じられないほど迷惑だからです。容器の外の火はランダムに消え、いつまで持つか教えてくれません。長い棒で小さな岩を壊すことができます。低木を調べることで採集でき、これにより液体用の容器を含む大量のゴミや廃棄物が手に入ります。ガラス/スチールのボトルや缶を見つけることができます。沸騰させて火を起こすことができれば、短期的には生き残れます。

薪は短期的には問題ではありません。植物や木、若い木からの長い棒など、何でも燃やすことができるからです。2、3本の長い棒で良い火が作れます。

簡易シェルターは睡眠を容易にし、掘削棒、サバイバル2、そしてたくさんの棒と松の大枝だけを必要とします。体温管理のために火の近くに配置してください。

## 天然資源

### 低木、食料の木、ベリーの茂み

低木は季節ごとに一度、ゴミを求めて採集できます。これは森で最も貴重な資源であり、唯一の鋼鉄の供給源です。季節によっては、春と夏にはカロリーとおそらく水分のために木のベリーに頼ることになりますが、低木は後でたくさんの食料を与えてくれます。

### 魚

魚を手に入れることができれば、優れたカロリー源になります。川と基本的な釣り竿が必要です。

### 若木

若木は長い棒を手に入れる方法であり、それを解体して重い棒にし、さらに割れた木にします。クラフトのヒントについては私の最初の1週間ガイドに従ってください。しかし一般的には、短い紐、石のナイフ、掘削棒、木製ビーズ、木の針、石の手斧、ウッドクロッグ、石のシャベル、石斧の順に進みます。これでサバイバル4と製作4になります。

### 草タイルとガマ

草は掘削棒やシャベルで取り除くことができ、わらの山が得られます。これはクラフト用の紐を作ることができます。ガマの植物繊維はぼろ布を作るために取っておくべきです。

### 粘土

粘土の地形は建設メニューで抽出する必要があります。大きな穴を掘ることで稀に見つかることがあります。たくさん必要になるので、川岸を見つけてください。

## 1週目の要件

### 食料保存

食料を保存するには、肉を燻製にするか、農作物を乾燥させるための炭焼窯と燻製ラックが必要です。調理2が必要なので、水を沸かして見つけた様々な食料を調理してください。これでおそらく訓練なしで到達します。窯は製作3で、土を掘る必要がありますが、即席シェルターを作ったならおそらく持っているでしょう。窯に木を入れて火をつけると、終わったら大量の木炭が手に入ります。

### 水の貯蔵

大量の水を貯蔵するには、多くの容器を採集するか、多くの粘土容器を作る必要があります。最も簡単なオプションは粘土の壺ですが、再密封できません。つまり、満タンの状態で持ち運ぶことはできません。

### 武器

木槍は最高の初期武器です。棍棒は実質的に無料で作れ、リーチ攻撃が使えない場合に適しています。革が手に入ればクォータースタッフにアップグレードできます。稼働する鍛冶場があれば、より良い武器がたくさん作れるので、これらは一時的なものです。

### 薪

薪は石の斧で木を丸太に切ることで得られます。実のなる木は取っておきたいかもしれません。

### 衣服

衣服は植物繊維で作られた布切れから作られます。編み針を手に入れてください。

## 長期目標

### シェルター

家を建てるのは時間がかかり、苦痛で、報われません。森に永遠に留まり、事前に生成された構造物や車に住まないことを選択した場合、最終的にはそれをやりたくなるでしょう。つるはしがあれば地面に住むことができますが、そうでなければ建設メニューで壁や床をいくつかチェックして、どの種類の家を建てるかを確認したいでしょう。ログハウスはかなり頑丈で安価ですが、常に火災のリスクがあります。モルタルは入手できないアイテムなので、石壁かアドビレンガに制限されるアドビモルタルで行きましょう。

### 稼働する鍛冶場

鍛冶場を稼働させることは重要です。石炉を作り、窯から木炭を得るのは簡単です。陶製るつぼ、彫金用タガネ、火ばさみ、ハンマーはすべて、適切な要件があれば簡単に作れます。金床は信じられないほど高価です。永遠に低木を探し続けることになるでしょう。もし都市を有効にしてプレイしているなら、車を粉々に壊すことができます。

### 車両

マップ生成によっては、本物の車をいじることに進むか、木製のカートや馬車で立ち往生することになります。車両の開始は建設メニューで行います。カヌーは良いアイデアです。木製フレームには釘と板が必要で、ほとんどの木製車両部品は木と釘だけが必要です。木製車両部品でメカニックを研鑽することになります。

### 電子工作

電子工作は、通常のマップ生成でプレイしている場合に限り、具体的にリストされています。あなたの世界にスポーンしない場合、運がありません。始める方法については通常のゲームプレイを参照してください。しかし、要点は、町で電化製品を解体し、懐中電灯や電動発火装置など、前進するためにできる低レベルのクラフトを作ることです。ソーラーパネル、取り付けられたバッテリー、オーブンが手に入れば、良い位置につけるでしょう。

> [!TIP]
>
> 都市に行かずにできることはこれくらいです。
