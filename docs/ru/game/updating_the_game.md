# Обновление игры

# Что нужно делать

0. Скачайте игру с https://github.com/cataclysmbn/Cataclysm-BN/releases
1. Распакуйте архив с игрой
2. Скопируйте папки `save` и `config` из старой директории игры в новую
3. (Опционально) Скопируйте папку `mods` из старой директории

# Чего делать нельзя

### Распаковывать новую версию поверх старой

Распаковка игры поверх старой директории может вызвать ошибки дублирования записей JSON. Если вы действительно хотите это сделать, обязательно удалите старую папку `data` перед распаковкой новой версии.

Иногда файлы данных удаляются из ядра игры. В новой версии игры этих файлов не будет, но распаковка архива не удаляет старые файлы. Старые файлы все равно будут прочитаны и загружены, что может привести к перезаписи новых записей.

### Хранить пользовательские моды в папке `data/mods`

Помещайте их непосредственно в папку `mods`, на том же уровне, что и `data`. Эта папка не существует по умолчанию, но игра будет читать её, если она есть.

Это позволит вам переносить моды между версиями игры, при этом позволяя "основным модам" обновляться корректно.
